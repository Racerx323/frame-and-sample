# Capture Cards

Capture card setup for frame stability and format compatibility.

## What To Capture

- Input format matrix (resolution, frame rate, chroma)
- Driver and firmware version
- Color-space and range assumptions
- Genlock or sync behavior where applicable

## Checklist

1. Match camera/source frame rate and card ingest settings.
2. Confirm no unwanted frame conversion in driver tools.
3. Validate long-run stability with dropped-frame counters.
