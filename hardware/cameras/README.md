# Cameras

Camera preparation for clean output and frame cadence reliability.

## What To Capture

- Clean output settings and overlays state
- HDMI/SDI format and frame rate
- Power and thermal behavior for long sessions
- Internal processing settings that affect latency

## Checklist

1. Set output cadence to target production frame rate.
2. Disable overlays and auto sleep behavior.
3. Verify output format is supported by capture chain.
