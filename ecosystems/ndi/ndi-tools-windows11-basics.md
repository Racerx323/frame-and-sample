# NDI Basics on Windows 11

## Download and install NDI Tools

1. Download the NDI Tools for Windows installer from the official NDI Tools page.
2. Run the installer and complete setup.
3. Reboot if Windows prompts you.

Reference: [Screen Capture HX | Docs and Guides](https://docs.ndi.video/all/using-ndi/ndi-tools/ndi-tools-for-windows/screen-capture-hx)

## Enable Screen Capture HX

1. Open NDI Launcher from the Start Menu.
2. In NDI Launcher, open the configuration gear and select Screen Capture HX.
3. Launch NDI Screen Capture HX.
4. Confirm the system tray icon appears.
5. Right-click the tray icon to adjust frame rate, capture selection, and audio source.

Quick check: Open Studio Monitor and verify the source is visible and stable.

## Enable NDI Webcam Input

1. From NDI Launcher, start Webcam Input.
2. Confirm the Webcam Input icon appears in the system tray.
3. Right-click the Webcam Input tray icon and choose the NDI source.
4. In Discord or another app, select the NDI virtual camera device.

Reference: [Webcam Input | Docs and Guides](https://docs.ndi.video/all/using-ndi/ndi-tools/ndi-tools-for-windows/webcam-input)

## Common fixes

- Ensure source and receiver are on the same network segment.
- If prompted by Windows Firewall, allow NDI Tools on Private networks.
