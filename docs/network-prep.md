# Network Preparation

Use this checklist for networked media transports, especially NDI and remote production paths.

## Baseline

- Use wired links for critical paths.
- Reserve capacity and avoid saturated uplinks.
- Keep switch firmware and NIC drivers updated.

## Validation

1. Measure throughput and packet loss under sustained load.
2. Validate latency and jitter across all media hops.
3. Confirm QoS policy does not starve control traffic.

## Documentation

- Record tested topology and VLAN assumptions.
- Record encoder/transport bitrate budget.
- Record known failure modes and fallback paths.
